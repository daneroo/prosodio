# Markdown

Authoring style. Tooling: [formatting.md](formatting.md).

- Unnumbered lists (`-`) and headings.
- Bullets over tables — except when directed or alignment is paramount.
- Bold/italics sparingly; no emojis unless asked.
- One `#` per file; fenced blocks have a language; end with one newline.
- Doc filenames use dashes (`CODING-STYLE.md`), not underscores.
- Prettier owns spacing and wrapping — don't manage it.
